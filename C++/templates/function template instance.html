# Function template instances

​ 并不像其它函数，编译器并不会直接编译函数模板，而是编译器遇到函数模板时，编译器会复制一份函数模板并且使用实际类型去代替类型占位符。有实际类型函数称之为函数模板实例。

​ 让我们来看一些这个过程。我们还是创建一个简单的函数模板：

template <typename T>
const T& max(const T& x, const T& y){
    return (x > y)? x:y;
}

​ 当编译器编译程序时，一旦遇到函数模板，

int x = 1, y = 2;
int result = max(x , y);

​ 编译器会复制这个函数模板，并且创建一个实例

const int& max(const int& x, const int& y){
    return (x > y)? x : y;
}

​ 那么，现在是一个正常的函数了，可以被编译为机器码了。如果不同的类型时，编译器又会重复这个过程。

# Operators, function calls, and function templates

​ 模板函数能够和内置类型(char, int, double, etc)和类一起使用。当编译器编译模板函数实例时，就像正常函数那样编译。在普通函数中，必须定义与类型一起使用的所有运算符或函数调用，否则会出现编译器错误。类似地，在函数模板中，任何操作符和函数调用必须为那些被实例化后的模板函数所定义。详情如下：

class Cents{
private:
    int m_cents;
public:
    Cents(int cents):m_cents{cents}{}
}

​ 如果我们想比较两个Cents类的大小，那么我们必须要定义操作符>的含义。

#include<iostream>
template <typename T>
const T& max(const T& x, const T& y){
    return (x > y)? x : y;
}
class Cents{
private:
    int m_cents;
public:
    Cents(int cents):m_cents{cents}{}
    friend bool operator>(const Cents& c1, const Cents& c2){
        return (c1.m_cents > c2.m_cents);
    }
    void print(){
        std::cout<<m_cents<<std::endl;
    }
}; 
int main(int argc, char* argv[]){
    Cents c1{2}, c2{3};
    Cents c3{max(c1, c2)};
    c3.print();
    return 0;
}

# Another example

​ 以下函数模板是被创建来计算在数组中元素的平均值。

#include<iostream>
template <typename T>	
T average(T *array, int length){
    T sum(0);
    for(int i = 0; i < length; i++){
        sum += array[i];
    }
    sum /= length;
    return sum;
}
class Cents{
private:
    int m_cents;
public:
    Cents(int cents):m_cents{cents}{}
    friend bool operator>(const Cents& c1, const Cents& c2){
        return (c1.m_cents > c2.m_cents);
    }
    friend std::ostream& operator<< (std::ostream &out, const Cents &cents)
    {
        out << cents.m_cents << " cents ";
        return out;
    }
    Cents& operator+=(const Cents &cents)
    {
        m_cents += cents.m_cents;
        return *this;
    }
    Cents& operator/=(int value)
    {
        m_cents /= value;
        return *this;
    }
};
int main()
{
    Cents array3[]{ Cents(5), Cents(10), Cents(15), Cents(14) };
    std::cout << average(array3, 4) << '\n';
 
    return 0;
}
